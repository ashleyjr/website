-- Updates --
The folder contains updates for the site.
Each file contains details of the update.
Each file should be called...

   <DATE>_<AREA>_<STATUS>.dat

Separated by underscores.

<DATE>:
   Code for the entyry date...
   4 digits for year
   2 digits for month 
   2 digits for day

<AREA>:
   w = website
   p = projects

<STATUS>:
   i = in progress
   f = finished
